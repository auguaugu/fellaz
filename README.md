## Branch

### `main`

- 실제 배포 되어있는 브랜치입니다.
- main 브랜치에 push 시 github action 으로 자동 배포됩니다.

### `feature`

- 개발 브랜치입니다.
- 페이지, 기능 단위로 나누어 작업합니다.

### `develop`

- 주 개발 브랜치입니다.
- main 브랜치에 merge/pr 은 항상 develop 브랜치만 이용합니다.
